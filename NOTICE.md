# Notice

`comptext-sparkctl` is an open-source, SPARK-adjacent research and demo repository for local-first evidence, validation, replay-oriented artifacts, and claim-hygiene workflows.

## Non-claims

This project is not an official BMDS/SPARK component.

This project does not provide:

- legal advice
- administrative decision automation
- regulatory certification
- EU AI Act certification
- official compliance guarantees
- official SPARK compatibility guarantees

The project should not be presented as an official public-sector decision system.

## Demo data

Hackathon and demo workflows should use synthetic data unless a separate approval explicitly allows another data source.

Do not include real citizen data, secrets, credentials, private keys, production tokens, or confidential administrative records in demo artifacts.

## Wording guidance

Preferred wording:

- SPARK-adjacent
- hackathon-oriented
- local-first evidence layer
- recorded workflow trace
- input/output validation
- hash-stable artifacts
- tamper-evident demo artifacts

Avoid claiming:

- official BMDS/SPARK status
- legal correctness
- autonomous administrative decisions
- certified compliance
- EU AI Act readiness
- reconstruction of private model reasoning
